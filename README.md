# BookHub

BookHub is a web application that allows users to discover, manage, and review books. Whether you're an avid reader looking to keep track of your reading list, a book enthusiast, or just someone who loves to share book recommendations with others, BookHub is the perfect platform for you.

- **User Authentication:** Create an account or log in to access the full range of features.
- **Book Management:** Upload books with details such as title, author, image, description, and mark them as "Read" or "Want to Read." Change the status as you progress.
- **Book Reviews:** Write reviews for books and rate them.
- **Genre-based Search:** Search for books by genre to discover new reading options.

## Getting Started

### Prerequisites

Before you start, ensure you have the following prerequisites installed:

- **Node.js**: You'll need Node.js to manage the client-side of the application. You can download it from [nodejs.org](https://nodejs.org/).

- **Python**: The server-side of the application uses Python. Ensure you have Python installed.

- **Pipenv**: Pipenv is used to manage Python dependencies. You can install it via pip:

  ```bash
  pip install pipenv

## Installation

1. Clone the repository
2. Navigate to the project directory and then into the server directory
3. Install the server-side dependencies using <code>pipenv install</code> and activate the virtual environment using <code>pipenv shell</code>
4. Run <code>python app.py</code> for port 5555
5. In another terminal, navigate to the project directory and then into the client directory
6. Run <code>npm install</code>
7. Run <code>npm start</code> for port 3000

## Usage

1. Register for a BookHub account or log in if you already have one.
2. Start by adding books to your library. Provide details like title, author, image, description, and select whether you've read it or want to read it.
3. Rate and review books you've read.
4. Explore books by genre using the search feature.
